# 前台统一视觉精修（M3-1）

## 目标

在 M2 完成信息层级收敛之后，本轮开始处理前台的第一批统一视觉问题，重点不是增加新功能，而是让已经明确的内容结构在视觉上更一致、更稳定。

## 本轮完成

### 1. 全局视觉基础补强

- 为 `apps/site/src/styles/global.css` 增加统一的卡片高光、按钮悬停、表单聚焦与细纹理背景
- 增加全局 `text-wrap`、`text-link` 箭头反馈与 `card-label-on-dark` 样式
- 把分散在页面内部的 `section-head-stack`、`section-note` 等通用视觉规则收敛到全局样式

### 2. 横向统一的页面模块

已统一以下模块的视觉行为：

- 浏览建议卡片
- 继续浏览卡片
- CTA 面板
- 信号/说明卡片

这些模块现在在成员、活动、文章等页面之间具有更一致的：

- 卡片背景层次
- 悬停反馈
- 标题颜色
- 栅格节奏

### 3. 修正真实视觉问题

本轮顺手修正了几类实际存在的问题：

- `card-label-on-dark` 之前只在首页生效，其他页面的深色标签样式并未真正统一
- 活动页与文章页中，部分浅底模块标题错误继承了深色 Hero 的浅色文字，导致对比度不理想
- 若干通用说明样式在不同页面里重复定义，后续维护成本偏高

## 当前结果

完成本轮后，前台已经从“页面各自可用”进入到“关键模块跨页面风格一致”的状态。

当前尤其更统一的部分包括：

- 成员 / 活动 / 文章三组页面的浏览引导模块
- 页面底部 CTA 区的结构与视觉层次
- 深色区块中的标签、按钮与说明文字对比度
- 表单与交互元素的悬停、聚焦反馈

## 下一轮建议

继续进入 `M3-2`，重点可放在：

1. 首页、关于我们、加入申请与成员/活动/文章页之间的节奏再统一
2. 分会董事会页的视觉密度优化
3. 在不增加复杂动画的前提下，补一轮更精细的进入动效与层次细节
